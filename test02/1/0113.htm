2. Tools

We use various operational review tools to assist in gathering and analyzing information about the organization. Sample tools include the initial survey form discussed previously and a Job Responsibilities Questionnaire to be filled in by each employee as shown in Exhibit 2.3. Both of these tools are examples of user- or client-provided data, which is to be encouraged, as this minimizes the data gathering time of the operational review team.
