Study Question

Select the best answer.

When can the analyst proceed in using an alternative approach to valuation instead of the use of guidelines?
 A. When another method can be shown to provide a more conservative value for the subject company
 B. When it has been demonstrated that an adequate number of guideline organizations cannot be found
 C. When the guideline companies used are major competitors
 D. When the valuator believes that it would be too difficult to search for potential guidelines

You must select an answer.
There is more than one correct answer.

Determining a conservative value might be beneficial to an actual buyer, but the value that the valuator reaches should be one that represents the standard of value fairly—not necessarily the most conservative one.

Major competitors might well be a reason to use comparables in a valuation, not a reason to use an alternative method.

Although the search for suitable guideline organization might be difficult, the business valuator still needs to make the effort to find enough suitable guideline firms.

The correct answer is B.

If an adequate number of guidelines cannot be found, an alternative method for the valuation can be employed. The valuator must be able to demonstrate that an adequate number of comparable organizations could not be found.
